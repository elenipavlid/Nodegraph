# Interactive Network Graph & Relationship Visualizer
## https://elenipavlid.github.io/NodeGraph/

An interactive network visualization web application built using vis-network. The application dynamically loads, parses, and maps entity relationship data directly from a cloud-hosted spreadsheet (Google Sheets API / CSV).
                                                               
## **Key Features**
- Dynamic Data Rendering: Real-time data pipeline that fetches live graph configurations, node connections, and edge metadata from a managed spreadsheet

- Interactive Graph Controls: Full-featured network navigation including targeted search, multi-layered filtering, and canvas state reset capabilities.

- Contextual Info Panel: Context-aware data display on hover/click, showcasing node attributes (e.g., Name, Category, Group) along with calculated network graph metrics.

- Pathfinding Integration: Graph analysis feature that automatically calculates and visually highlights the optimal path to the farthest connected node in the network.

- Physics Toggle & Canvas Optimization: User control to enable/disable physics simulation to freeze node layouts, improving UX and performance on large datasets.
